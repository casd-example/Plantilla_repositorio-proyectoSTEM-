¿Cómo usar el editor de bloques?
  1. Selecciona un proyecto del repositorio a la izquierda.
  2. Usa la pestaña Ejecutar para ver el simulador funcionando.
  3. Usa la pestaña Editor Bloques para abrir el editor completo de MakeCode en esta ventana.
  4. Para agregar tu propio proyecto: copia el enlace desde MakeCode o Scratch y pégalo en el campo de la izquierda.
  5. Clic en Abrir completo para editar en pantalla completa.
💡 Consejo: En MakeCode Micro:bit ve a Compartir → Publicar proyecto y copia el enlace para pegarlo aquí.
